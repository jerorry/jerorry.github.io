hexo安装备忘

hexo是一个博客系统

安装需要：
nodejs
git

nodejs安装：
官网下载，安装完成

git安装：
官网下载，安装完成

hexo安装

cmd终端执行：
1，npm install -g hexo-cli
2，npm install hexo-server –save
cd
3，切换到 e:
npm install hexo-cli -g

4，初始化项目：hexo init blog

5，启动项目
cd jerorry
npm install
hexo server

6，部署
以git部署，需要安装 npm install hexo-deployer-git –save
部署配置：
deploy:
type: git
repo: https://jerorry:s801218@github.com/jerorry/jerorry.github.io.git
branch: master
